# VPN Project

This is a basic VPN implementation in Python, showcasing a client-server architecture.

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/bardia5008/C303030.git
   cd C303030
   ```

2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the server:
   ```bash
   python vpn_server.py
   ```

4. Run the client:
   ```bash
   python vpn_client.py
   ```
